# Portfolio Website Project

## Overview
Simple, elegant portfolio website for Rayyan, an upcoming AI specialist. Built with vanilla HTML/CSS/JS for maximum simplicity and easy hosting on GitHub Pages.

## Project Structure
```
portfolio/
├── index.html          # Main HTML file with all sections
├── styles.css          # Styling and responsive design
├── script.js           # Smooth scrolling and interactions
├── README.md           # Setup and deployment instructions
└── CLAUDE.md           # This file - project documentation
```

## Design Philosophy
- **Recruiter-friendly**: Clear hierarchy, easy to scan, professional presentation
- **Simple & elegant**: No frameworks, no build process, just clean code
- **Responsive**: Works seamlessly on desktop, tablet, and mobile
- **Performance**: Fast loading with minimal dependencies

## Sections
1. **Hero** - Name, title, tagline with gradient background
2. **About** - Brief bio highlighting AI focus
3. **Skills** - Grid layout showcasing ML/AI technologies
4. **Projects** - Card-based project showcase with tech tags
5. **Experience/Education** - Timeline format
6. **Contact** - Email, LinkedIn, GitHub links

## Tech Stack
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- No external dependencies

## Key Features
- Fixed navigation bar with smooth scrolling
- Fade-in animations on scroll
- Hover effects on cards and buttons
- Mobile-responsive navigation
- Semantic HTML structure

## Deployment
Optimized for GitHub Pages:
1. Push to repository named `username.github.io`
2. Site automatically deploys to `https://username.github.io`
3. No build step required

## Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Uses standard CSS features (Grid, Flexbox, CSS Variables)
- Smooth scrolling with fallback
- Intersection Observer for animations
